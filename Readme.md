# Movie Management API

## End Points
### Get all Movies
Method : ```GET```
```bash
/movies
```
### Get Movies by Id
Method : ```GET```
```bash
/movies/id
```
### Create new Movie
Method : ```POST```
```bash
/movies
```
### Update Movie
Method : ```PUT```
```bash
/movies/id
```
### Delete Movie
Method : ```DELETE```
```bash
/movies/id
```
